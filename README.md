# Loan Default Prediction

## Objective
Predict whether a loan application will be approved using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Algorithm
- Random Forest Classifier

## Workflow
1. Data Loading
2. Missing Value Handling
3. Label Encoding
4. Data Visualization
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation
9. Model Saving

## Results
- Accuracy: 78.05%

## Outputs
- Loan Approval Prediction
- Confusion Matrix
- Classification Report
- Saved Model

## How to Run
1. Install requirements
2. Open loan_default_prediction.ipynb
3. Run all cells
